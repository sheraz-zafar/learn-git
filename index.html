Introduction to Git & GitHub

Understanding the fundamentals of version control

What is Git?

Git is a distributed version control system that tracks changes in your code. Think of it as a time machine for your projects.

  • Track file changes
  • Collaborate with others
  • Revert to previous versions

What is GitHub?

GitHub is a cloud platform that hosts Git repositories and provides collaboration tools for developers worldwide.

  • Host repositories online
  • Code review tools
  • Project management

Why Learn Git?

Git is essential for modern software development and is used by millions of developers worldwide.

  • Industry standard
  • Team collaboration
  • Open source contribution

Git Basics

Installation & Setup

Installation Commands 
# Download Git from git-scm.com
# Configure your identity
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

# Verify installation
git --version

Basic Commands

git init

Initialize a new Git repository

git init

git status

Check the status of your repository

git status

git add

Stage files for commit

git add .

git commit

Save staged changes

git commit -m "Your message"

git log

View commit history

git log

git remote

Connect to GitHub repository

git remote add origin <repo-url>
